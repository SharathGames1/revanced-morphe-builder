Patched APKs Builder
================
This project is based on `jhc's patcher <https://github.com/j-hc/revanced-magisk-module>`__ for revanced patches

Some code is taken with help of Generitive AI such as replacing filters and entire new pipeline for  basic apk edits

Credits to https://github.com/FiorenMas/Revanced-And-Revanced-Extended-Non-Root/tree/main for user agent code
